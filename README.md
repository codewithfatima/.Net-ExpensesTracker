# Expense Tracker

A simple web application to manage your daily expenses, built with **Angular**.

## Features
- User Registration and Authentication.
- Securely track expenses using browser local storage.
- Add, edit, and delete expense entries.

## Getting Started
1. Install dependencies: `npm install`
2. Run the development server: `ng serve`
3. Navigate to `http://localhost:4200/`

## How it works
This app uses Angular Services and RxJS Observables to manage user authentication state across the entire application.
